typeof

You can use the typeof operator to find out the type of a value

      typeof 10
      << "number"

String Properties and Methods

string.length

The .length property will return the length of the string as an integer

      "Mary".length
      << 4

string.toUpperCase()

The .toUpperCase() method returns a string in all uppercase letters

      let name = "mary"
      name.toUpperCase()
      << MARY

string.toLowerCase()

The .toLowerCase() method returns a string in all lowercase letters

      name.toLowerCase()
      << mary

chartAt()

To find which character is at a specified position in the a string, use the charAt() method

      name.charAt(2)
      << "r"

indexOf()

The indexOf() method will tell you the index of a specified character

      name.indexOf('r')
      << 2

If the specified character is not found in the string, the method will return -1

      name.indexOf('p')
      << -1

lastIndexOf()

The lastIndexOf() method is similar to the indexOf() method, but this one starts from the end of the string

      let llama = 'llama'
      llama.lastIndexOf('l')
      << 1

Compare indexOf() and lastIndexOf(). Remember that indexOf() starts looking at the start of the string

      llama.indexOf('l') 
      << 0

These methods are case-sensitive. 

      llama.indexOf('L') 
      << -1

concat()

The concat() method will concatencate two strings

      'Hello'.concat('World')
      << HelloWorld

You can chain more than one string. Here we concatencate a space between 'Hello' and 'World'

      'Hello'.concat(' ').concat('World')
      << Hello World

You can even chain unrelated methods

      'Hello'.concat(' ').concat('World').toUpperCase()
      << HELLO WORLD

trim()

The trim() method will remove whitespace from the beginning and end of the string

      "  Hello      ".trim();
      << "Hello"

Number Properties and Methods

Both integers and floating point numbers are of type integer

toExponential()

When using dot notation with numbers, it is best to use two dots to prevent confusion with decimal points

      5..toExponential()
      << 5e+0
    
      5 .toExponential()
      << 5e+0
    
      5.0.toExponential()
      << 5e+0
    
      (5).toExponential()
      << 5e+0

toFixed()

Use the .toFixed() method to control the number of decimal places. This will return a string.

      3.1415926.toFixed(3)
      3.142

toPrecision()

The toPrecision() method rounds a number to a fixed number of significant figures and returns it as a string.

      3.1315926.toPrecision(3)
      << 3.13

Parsing Numbers

The parseInt() and parseFloat() functions can parse a string into an integer or a float, respectively

parseInt(number, { optional: base })

      parseInt("10", 10)  // 10
      parseInt("10", 2)  // 2

If the string starts with a number and contains text, it will only attempt to parse the numbers until it finds a letter

      let address = '123 Main St'
      parseInt(address)
      << 123

Using parseInt() on a decimal will truncate everything after the decimal

      parseInt(3.14)
      << 3

parseFloat() works similar to parseInt()

      parseFloat("2.9")
      2.9
